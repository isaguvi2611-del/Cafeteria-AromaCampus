# Cafeteria 

## AromaCampus

Codigo creado con el fin de simular una cafeteria donde se pueda ver un menu de opciones y agregar mas cafes al menu

## ¿Como esta dividido?

tiene tres archivos llamados:
- agregarCafe.py
- main.py
- menu.py

## AgregarCafe
en este archivo podemos encontrar las opciones de si desea o no agregar un cafe y una funcion para agregar un cafe

### Main.py
 aqui podemos encontrar el menu principal donde se puede elegir entre ver el menu o agregar un nuevo producto

### manu.py
Aqui podemos encontrar el menu ya establecido con sus respectivos precios
